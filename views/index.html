Request Header Parser Microservice

Description

This microservice will get you the IP address, preferred languages (from header Accept-Language)
and system infos (from header User-Agent) of your device.

Usage

Generate info

Click or copy/paste this link below in the browser to generate the info.
https://ek-request-header-parser-microservice.glitch.me//api/whoami

Example Output:

{
"IP Address":"14x.105.8.2Y6",
"Language":"en-US",
"Software":"Windows NT 10.0; Win64; x64"}
